N8RO Character Animation Project

This project demonstrates a closed-library joint-angle controller
integrated into the N8RO runtime environment.

Implemented Features:
- 10 skeletal joint rotations
- Runtime pose manipulation
- Real-time animation override
- GLB human model visualization
- Joint-angle based motion control

Implemented States:
- Idle
- Walk

Controlled Joints:
1. Left Ankle
2. Right Ankle
3. Left Knee
4. Right Knee
5. Left Hip
6. Right Hip
7. Left Shoulder
8. Right Shoulder
9. Left Elbow
10. Right Elbow

The plugin controls 10 joints and demonstrates idle and walk motion states
